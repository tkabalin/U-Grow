Setting up the Program

The database must be stored in the same folder as the program with the name: UGrow_Database.mdb. See the structure below.

Product images added to the folder Product_Images must be of the file extension .jpg, otherwise they will not be loaded. Their file name should be the ProductID of the product. The optimal aspect ratio for product images is 5:4, so images should be cropped to avoid stretching.

Receipts are stored in a text file with the name: Receipt.txt in the same folder as the program.

Seller logins must be added to the text file with the name: Seller_Logins.txt in the same folder as the program, in the format username#password, with the username not exceeding 128 characters and the password not exceeding 128 characters.

The date format of the computer must be set to dd/mm/yyyy, otherwise the receipt system may not function properly.

The program is designed for a 1080p screen, but functions properly on any resolution down to 720p. Resolutions above or below that may make the program unusable.

Format of Various IDs

CustomerID: first letter of the first name, first two letters of the last name and three random digits

ProductID: first three letters of product name and 3 random digits

OrderID: the letters INV and five random digits

OrderProdID: sequential (auto) number
